# Writer APIs

The Directory Writer APIs are in the `aserto.directory.writer.v3.Writer` service.

- `SetObject` - create or update an object by `type` and `identifier`
- `DeleteObject` - delete an object by `type` and `identifier`

- `SetRelation` - create or update a relation by its object, relation, and subject
- `DeleteRelation` - delete a relation by its object, relation, and subject
